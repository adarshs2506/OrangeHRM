# OrangeHRM Testing Project

## Overview:

This project focuses on manual testing of the OrangeHRM web application. 
The objective is to ensure that all HR management functionalities work correctly as per requirements and deliver a smooth user experience.

## Objectives
* Validate end-to-end HR workflows
* Identify functional and UI defects
* Ensure proper role-based access control
* Verify data validation and security checks
* Improve overall application quality and stability

## Modules Covered
* Login
* HR Administration
* PIM (Employee Management)
* Reports and Analytics
* Time / Attendance
* Recruitment
* Performance
* Dashboard
* Logout
  
## Testing Types Performed
* Functional Testing
* UI Testing
* Regression Testing
* Smoke Testing
* Sanity Testing
* System Testing
* Security Testing (SQL Injection, Session Handling)
